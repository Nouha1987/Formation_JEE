<li class="dropdown"> <a class="dropdown-toggle" data-toggle="dropdown" href="@{/}">Question & Réponse <span class="caret"></span></a> <ul class="dropdown-menu"> <li><a href="page1">Page 1</a></li> <li><a href="page2">Page 2</a></li> <li><a href="page3">Page 3</a></li> </ul> </li>
Remarque
Il faut faire l'appel à "bootstrap-dropdown.js"
Le chemin dans <a class="dropdown-toggle" data-toggle="dropdown" href="@{/}"> doit être le lien principale puis dans les filles toogle il contient la suite du lien
Exemple : le lien est /test/page1 dans le lien principale on met /test
et dans les fille on met page1
